# ASAPP Front End Challenge

## Run Locally

Clone the project

```bash
  git clone git@github.com:iocevelasco/asapp-frontend.git
```

Go to the project directory

```bash
  cd asapp-frontend
```

Install dependencies

```bash
  cd api npm install 
  cd frontend npm install 
```

Start Pr

```bash
  npm run start-dev in the root to run in parallel
```
